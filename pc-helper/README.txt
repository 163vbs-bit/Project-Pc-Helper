System Control Panel 1.0

Краткое описание
Простая панель управления компьютером на Python и Tkinter. Программа открывает популярные ресурсы, запускает проводник и выполняет системные действия после подтверждения пользователя.

Файлы версии
- 2пр.py - исходный код приложения.
- 2пр.exe - собранная версия для запуска без Python.
- Пользователь.png - блок схема программы со стороны пользователя
- Внутренняя.png - блок схема программы со стороны программы
- СхемаПрограммы.drawio - редактируемая диаграмма draw.io.
- README.txt - описание версии.

Требования для запуска исходника
- Python 3.6 или новее.
- Стандартные библиотеки Python: tkinter, webbrowser, os, platform, subprocess.
- Для запуска exe Python не нужен.

Запуск
1. Через Python:
   python 2пр.py

2. Через exe:
   запустить файл 2пр.exe.

Интерфейс
После запуска открывается окно "Панель управления" размером 400x400. В окне есть заголовок, шесть кнопок и строка состояния.

Кнопки
- Открыть браузер - открывает https://google.com через webbrowser.open.
- Открыть YouTube - открывает https://youtube.com через webbrowser.open.
- Открыть проводник - открывает текущую папку проекта.
- Перезагрузить компьютер - спрашивает подтверждение и выполняет команду перезагрузки.
- Выключить компьютер - спрашивает подтверждение и выполняет команду выключения.
- Выход - закрывает окно приложения через root.destroy().

Внутренняя логика
1. Импортируются стандартные библиотеки.
2. Создается главное окно Tkinter.
3. Создаются кнопки и привязываются к функциям.
4. Пользователь нажимает кнопку.
5. Вызывается соответствующая функция.
6. Для системных действий сначала показывается диалог подтверждения.
7. Программа выбирает команду по операционной системе: Windows, macOS или Linux.

Права доступа
Для выключения и перезагрузки могут потребоваться права администратора. Если команда не выполнится, программа покажет сообщение об ошибке.

Горячие клавиши
В версии 1.0 горячие клавиши не используются.

System Control Panel 2.0

Краткое описание
Расширенная версия панели управления компьютером на Python и Tkinter. По сравнению с версией 1.0 приложение оформлено как класс, имеет темную тему, дополнительные кнопки, вывод информации о системе, проверку прав администратора и горячие клавиши.

Файлы версии
- 2прUpdate.py - исходный код приложения.
- 2прUpdate.exe - собранная версия приложения.
- Пользователь2 - блок схема программы пользователя
- Внутренняя2 - блок схема программы 
- СхемаПрограммы2.drawio - редактируемая диаграмма draw.io.
- README.txt - описание версии.

Требования для запуска исходника
- Python 3.6 или новее.
- Стандартные библиотеки Python: tkinter, webbrowser, os, platform, subprocess, sys.
- На Windows для проверки прав администратора используется ctypes.
- Для запуска exe Python не нужен.

Запуск
1. Через Python:
   python 2прUpdate.py

2. Через exe:
   запустить файл 2прUpdate.exe.

Интерфейс
После запуска открывается окно "System Control Panel" размером 800x500. В окне есть заголовок, три ряда кнопок, блок информации о системе и нижняя строка состояния.

Кнопки
- Открыть браузер - открывает https://www.google.com.
- Открыть YouTube - открывает https://www.youtube.com.
- Открыть проводник - открывает текущую папку.
- Выключить компьютер - проверяет права администратора, спрашивает подтверждение и выполняет выключение.
- Перезагрузить компьютер - проверяет права администратора, спрашивает подтверждение и выполняет перезагрузку.
- Выход из программы - спрашивает подтверждение и закрывает окно.
- Блокнот - запускает стандартный текстовый редактор.
- Калькулятор - запускает стандартный калькулятор.
- GitHub - открывает https://github.com.

Горячие клавиши
- Ctrl+B - открыть браузер.
- Ctrl+Y - открыть YouTube.
- Ctrl+Q - выйти из программы.
- Escape - выйти из программы.

Внутренняя логика
1. main() проверяет права администратора на Windows.
2. Создается объект SystemControlPanel.
3. setup_window() настраивает окно и цвета.
4. create_widgets() создает заголовок, кнопки, информацию о системе и строку состояния.
5. setup_bindings() подключает горячие клавиши.
6. При нажатии кнопки вызывается метод класса.
7. Метод выполняет действие, обновляет статус или показывает ошибку.
8. Опасные операции выключения и перезагрузки требуют подтверждения.

Отличия от версии 1.0
- Код организован в класс.
- Добавлены Блокнот, Калькулятор и GitHub.
- Добавлена проверка прав администратора.
- Добавлены горячие клавиши.
- Добавлен вывод информации о системе.
- Строка состояния обновляется после действий.

Права доступа
Если прав администратора нет, выключение и перезагрузка блокируются предупреждением. На Windows при запуске программа пытается запросить права администратора.
